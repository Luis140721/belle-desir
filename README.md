# Belle Désir

## Estructura
- `/backend` — API REST (Node.js + Express + TypeScript + Prisma)
- `/frontend` — Landing y tienda (HTML + CSS + JS vanilla)

## Levantar en desarrollo
```bash
# Backend
cd backend && npm install && npm run dev

# Frontend
cd frontend && npx serve . 
# o abre index.html directamente en el navegador
```
